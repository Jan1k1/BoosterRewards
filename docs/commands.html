Commands

Player Commands

Command Alias Description Permission
/link [code] None Link your Minecraft account to Discord None
/unlink /logout Unlink your Discord account None
/claim None Claim any pending rewards None
/br info None View your current boost status None

Admin Commands

Main Command: /boosterrewards or /br

Command Description Permission
/br reload Reload all config files boosterrewards.admin
/br admin Open the admin GUI boosterrewards.admin
/br sync [player] Force reward sync for a player boosterrewards.admin
/br forceunlink <player> Force unlink a player's account boosterrewards.admin

Discord Commands

Slash commands available in your Discord server.

Command Description
/link Get a code to link your account
/unlink Unlink your current Minecraft account
/status Check your current boost status and rewards
