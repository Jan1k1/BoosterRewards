Basic Setup Examples

LuckPerms Rank Reward

The most common setup. Give a player a LuckPerms group when they boost.

rank-reward.yml
rewards:
  booster:
    enabled: true
    on-boost:
      - "lp user %player% parent add booster"
      - "msg %player% &dYou have received the Booster rank!"
    on-stop:
      - "lp user %player% parent remove booster"
      - "msg %player% &cYour Booster rank has expired."

Item Rewards

Give physical items using the give command.

item-rewards.yml
rewards:
  booster:
    enabled: true
    on-boost:
      # Give 64 diamonds
      - "give %player% diamond 64"
      # Give a custom renamed item (using essentials/cmi command)
      - "give %player% diamond_sword 1 name:&dBooster_Sword"

Economy Rewards

Give money using Vault/Essentials economy.

economy-rewards.yml
rewards:
  booster:
    enabled: true
    on-boost:
      - "eco give %player% 5000"
      - "broadcast &d%player% just boosted the server!"
