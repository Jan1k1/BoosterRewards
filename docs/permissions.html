Permissions

Standard Permissions

Permission Description Recommended For
boosterrewards.link Allows linking account Everyone (default)
boosterrewards.unlink Allows unlinking account Everyone (default)
boosterrewards.claim Allows claiming rewards Everyone (default)

Admin Permissions

Permission Description Recommended For
boosterrewards.admin Full access to all commands Admins / Ops
boosterrewards.reload Allows reloading config Admins
boosterrewards.gui Access to admin GUI Admins

Reward Permissions

You can define custom permissions for specific reward tiers in config.yml.

rewards:
  booster:
    # Requires permission: boosterrewards.booster
    permission: "boosterrewards.booster"
Wildcard: boosterrewards.* grants all permissions.
