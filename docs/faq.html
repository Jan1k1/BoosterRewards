Frequently Asked Questions

Does this work on BungeeCord/Velocity?

The plugin is designed for Spigot/Paper backend servers. If you want rewards to sync across a network, use a MySQL database in config.yml and install the plugin on all backend servers.

Can I change the linking code length?

Yes! In config.yml, change linking.code-length.

Do rewards disappear when I stop boosting?

Yes, if you configure the on-stop section in config.yml. You can remove roles/ranks or take money away.

Is this plugin free?

Yes, BoosterRewards is 100% free and open source.

Where can I get support?

Join our Discord Server or open an issue on GitHub.
