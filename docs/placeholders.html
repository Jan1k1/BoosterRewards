Placeholders

These placeholders can be used in reward commands and messages.

Player Placeholders

Placeholder Description Example
%player% Minecraft username Notch
%uuid% Player UUID 069a79f4-44e9-4726-a5be-fca90e38aaf5

Discord Placeholders

Placeholder Description Example
%discord_user% Discord username user#0000
%discord_id% Discord User ID 123456789012345678

Boost Placeholders

Placeholder Description Example
%boost_start% Date boost started 2026-02-07 14:30:00
%boost_duration% Duration of boost 2 weeks, 3 days
%tier% Reward tier name booster

PlaceholderAPI Support

We provide standard PlaceholderAPI expansions for use in other plugins (tab, scoreboard, etc).

Placeholder Description
%boosterrewards_is_boosting% Returns "true" or "false"
%boosterrewards_discord_user% Linked Discord username
%boosterrewards_boost_duration% Readable duration string
