BoosterRewards Documentation

Reward Discord server boosters with automatic in-game perks

Installation

Step 1: Create Discord Bot

  1. Go to Discord Developer Portal
  2. Click New Application
  3. Go to Bot tab → Reset Token
  4. Enable Server Members Intent under Privileged Gateway Intents
  5. Go to OAuth2 → URL Generator
  6. Select scopes: bot, applications.commands
  7. Select permissions: Send Messages, Embed Links
  8. Copy the URL and invite bot to your server

Step 2: Install Plugin

  1. Download the latest JAR from Releases
  2. Place in plugins/ folder
  3. Start server to generate config files
  4. Stop server

Step 3: Configure

Edit plugins/BoosterRewards/discord.yml:

bot:
  token: "YOUR_BOT_TOKEN_HERE"
  guild-id: "YOUR_SERVER_ID"

Edit plugins/BoosterRewards/config.yml:

rewards:
  booster:
    enabled: true
    on-boost:
      - "lp user %player% parent add booster"
      - "give %player% diamond 5"
    on-stop:
      - "lp user %player% parent remove booster"

Step 4: Start Server

Start your server. Check console for successful connection:

[BoosterRewards] Connected to Discord guild: YourServerName

Configuration

Database Settings

database:
  type: H2              # H2, MYSQL, or MARIADB
  host: "localhost"
  port: 3306
  database: "boosterrewards"
  username: "root"
  password: ""
  pool-size: 10

H2 - File-based, no setup required (good for small servers)

MySQL/MariaDB - External database (recommended for networks)

Reward Configuration

rewards:
  booster:
    enabled: true
    permission: "boosterrewards.booster"
    on-boost:
      - "lp user %player% parent add booster"
      - "give %player% diamond 5"
      - "eco give %player% 1000"
    on-stop:
      - "lp user %player% parent remove booster"
    cooldown: 0
    one-time: false

Command Prefixes

  • console: or no prefix - Run as console
  • player: - Run as the player

Available Placeholders

Placeholder Description
%player% Minecraft username
%uuid% Player UUID
%discord_user% Discord username
%discord_id% Discord user ID
%boost_start% When boost started
%boost_duration% How long boosting

Sync Settings

sync:
  interval: 300                    # Check every 5 minutes
  remove-rewards-on-stop: true     # Auto-remove rewards
  check-on-join: true              # Check on player join
  verify-on-startup: true          # Sync on server start

Linking Settings

linking:
  mode: DISCORD_TO_MINECRAFT       # or MINECRAFT_TO_DISCORD
  code-expiry: 300                 # 5 minutes
  code-length: 6
  allow-relink: true

DISCORD_TO_MINECRAFT: `/link` in Discord → code → `/link ` in Minecraft

MINECRAFT_TO_DISCORD: `/link` in Minecraft → code → `/link ` in Discord

Troubleshooting

Bot Not Connecting

  • Verify bot token in discord.yml
  • Enable "Server Members Intent" in Developer Portal
  • Check guild-id is correct
  • Check console for errors

Rewards Not Giving

  • Enable debug-mode: true in config
  • Test commands manually in console
  • Verify player is linked with /status in Discord
  • Check reward tier is enabled

Link Codes Not Working

  • Codes expire after 5 minutes (default)
  • Verify linking.mode matches your workflow
  • Ensure no typos (case-sensitive)
  • Check bot has permission to send DMs
