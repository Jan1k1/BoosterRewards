BoosterRewards

Version: 0.0.1

Minecraft: 1.21.4+ (Paper/Spigot)

Java: 21+

Overview

BoosterRewards is a Minecraft plugin that automatically rewards Discord server boosters with in-game perks. Link Minecraft accounts to Discord, automatically sync boost status, and execute custom commands when players boost or stop boosting your server.

Features

  • Account Linking - Secure 6-digit code system for linking Minecraft to Discord
  • Automatic Synchronization - Checks boost status every 5 minutes (configurable)
  • Custom Rewards - Run any command when players boost (permissions, items, currency)
  • Automatic Revocation - Remove rewards when boost expires
  • Multi-Tier Support - Different rewards for different boost levels
  • Performance Optimized - Async operations, in-memory caching, zero lag
  • Flexible Database - H2 (embedded) or MySQL/MariaDB for networks
  • Admin GUI - Visual interface for managing rewards
