Setup in 5 Minutes

1

Create Discord Bot

Go to discord.com/developers ‚Üí New Application ‚Üí Bot tab ‚Üí Copy token

Enable "Server Members Intent" under Privileged Gateway Intents

2

Install Plugin

Download JAR ‚Üí place in plugins/ ‚Üí start server ‚Üí stop server

3

Configure

Edit discord.yml - add bot token and server ID

Edit config.yml - set up rewards

4

Start Server

Bot will connect to Discord. Players can now use /link

Reward Example

rewards:
  booster:
    enabled: true
    on-boost:
      - "lp user %player% parent add booster"
      - "give %player% diamond 5"
    on-stop:
      - "lp user %player% parent remove booster"
