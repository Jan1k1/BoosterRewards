Messages Configuration (messages.yml)

Customize all in-game messages sent to players. Supports legacy color codes (&a) and Hex colors (&#RRGGBB).

General Settings

# Prefix shown before most plugin messages
prefix: "&#fb00ff&lN&#fc45ff&li&#fd6aff<&#fe86ff&lr&#ff9fff&lo 󙀸» &7"

# Error messages
no-permission: "&cYou don't have permission to do that."
player-only: "&cThis command can only be executed by players."
invalid-args: "&cInvalid usage! Use: &f%usage%"

Linking Messages

linking:
  code-generated: "&aRun &n/link %code%&a in Discord to finish!"
  code-expired: "&cYour linking code has expired."
  success: "&aSuccessfully linked your account to &f%discord_user%&a!"
  already-linked: "&cYou are already linked to a Discord account."
  not-linked: "&cYou must link your account first! Use /link."
  unlinked: "&aSuccessfully unlinked your account."

Reward Messages

rewards:
  received: "&aYou received the &f%tier%&a reward!"
  expired: "&cYour booster reward has expired."
  claimed: "&aYou have claimed your pending rewards!"
  none-pending: "&cYou don't have any pending rewards."
Tip: Set any message to "" (empty string) to disable it completely.
