Quick Start Guide

Get up and running in under 5 minutes.

1. create Bot

  • Create bot at Discord Dev Portal.
  • Enable "Server Members Intent".
  • Invite to server with bot and applications.commands.

2. Install

  • Drop JAR in plugins/.
  • Restart server.

3. Configure

  • Put Token & ID in discord.yml.
  • Set rewards in config.yml.
  • Restart server.

4. Test

  • Run /link in Discord.
  • Run /link <code> in Minecraft.
  • Boost server and check rewards!
