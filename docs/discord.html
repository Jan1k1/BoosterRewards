Discord Configuration (discord.yml)

The discord.yml file contains sensitive information for connecting to your Discord bot.

Security Warning: Never share your discord.yml file or your bot token. Any user with your token can control your bot.

Bot Credentails

bot:
  # Your Discord Bot Token
  token: "OTk5..."
  
  # Your Discord Server ID (Right click server -> Copy ID)
  guild-id: "123456789012345678"
  
  # Activity to show on the bot
  activity: "Watching Server Boosters"
  
  # Status: ONLINE, IDLE, DND, INVISIBLE
  status: ONLINE

Channel Settings

channels:
  # Channel to send boost logs/announcements
  logs: "123456789012345678"
  
  # Channel to send database errors (admin only)
  errors: "123456789012345678"

Role Settings

roles:
  # Role ID given to linked players (optional)
  linked: "123456789012345678"
  
  # Role ID for boosters (used to verify status)
  booster: "123456789012345678"
