Troubleshooting

Bot Not Connecting

Error: [JDA] Login Exception: The provided token is invalid!
  • Cause: The token in discord.yml is incorrect.
  • Fix: Regenerate your token in the Discord Developer Portal and update discord.yml.
Error: [JDA] GatewayIntent.GUILD_MEMBERS is not enabled!
  • Cause: "Server Members Intent" is disabled.
  • Fix: Go to Developer Portal → Bot → Privileged Gateway Intents → Enable "Server Members Intent".

Slash Commands Not Appearing

  • Cause: Bot missing permissions or scopes.
  • Fix: Ensure you invited the bot with applications.commands scope. Kick the bot and re-invite using the correct link.

Rewards Not Giving

  • Check console for errors when you boost.
  • Ensure rewards.booster.enabled is set to true in config.yml.
  • Verify your link status with /status in Discord.
  • Check if you have permission boosterrewards.claim.

Database Errors

Error: SQLSyntaxErrorException: Table 'boosterrewards' doesn't exist
  • Cause: Database not initialized or corrupt.
  • Fix: Delete database/boosterrewards.db (if using H2) or drop tables (if MySQL) and restart server.
